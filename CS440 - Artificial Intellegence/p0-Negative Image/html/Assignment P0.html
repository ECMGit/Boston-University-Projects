Problem

Using a live video input, invert the colors and display in real time.

Implementation

Skeleton code was provided to capture video from a webcam. The inverted image was computed by inverting the color of each pixel in the image. Nested loops were used to access each pixel. To invert to color, the original color intensity for each channel (red, gree, and blue) was subtracted from 255 (the highest intesity value).
